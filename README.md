# \## 🔍 About This Project

# 

# This project builds a complete \*\*Sales \& Demand Forecasting System\*\* using Machine Learning.  

# It predicts future daily sales based on 2 years of historical retail data.

# 

# \---

# 

# \## 🎯 What It Does

# 

# \- ✅ Generates realistic retail sales data (with trends \& seasonality)

# \- ✅ Cleans data and handles missing values

# \- ✅ Creates time-based features (day, week, month, lag features)

# \- ✅ Trains 3 ML models and picks the best one

# \- ✅ Forecasts the next 30 days of sales

# \- ✅ Produces a business-friendly dashboard with 5 charts

# 

# \---

#

# | Pandas | Data manipulation |

# | Scikit-learn | ML models |

# | Matplotlib / Seaborn | Visualizations |

#

# 

# \## 📊 ML Models Used

# 

# | Model | Description |

# |-------|-------------|

# | Random Forest | Best performer (ensemble) |

# | Gradient Boosting | Strong alternative |
